🧠 THUẬT TOÁN AI

5 Thuật toán Tìm kiếm

Thuật toánLoạiĐặc điểmCấu trúc
BFSUninformed Duyệt theo lớp, đường ngắn nhất (số bước)Queue (FIFO)
DFSUninformed Đi sâu trước, không đảm bảo tối ưuStack (LIFO)
UCSUninformed Chi phí nhỏ nhất, tối ưu khi trọng số khác nhauPriority Queue
A*Informed f(n) = g(n) + h(n) — tối ưu nếu h admissiblePriority Queue
GreedyInformed f(n) = h(n) — nhanh, không đảm bảo tối ưuPriority Queue

📏 Heuristic: Manhattan |x₁−x₂| + |y₁−y₂|   hoặc   Euclidean √((x₁−x₂)² + (y₁−y₂)²)

SLIDE 15
📐 PID CONTROLLER

PID Line Following

// PID Line Following float error = calculateLineError(sensors[5]); float P = Kp * error; float I = Ki * integral; float D = Kd * (error - prevError) / dt; float correction = P + I + D; int speedL = BASE_SPEED + correction; int speedR = BASE_SPEED - correction; setMotor(LEFT, constrain(speedL, 0, 255)); setMotor(RIGHT, constrain(speedR, 0, 255));

Công thức PID

correction = Kp×e + Ki×∫e·dt + Kd×de/dt

Tinh chỉnh Ziegler-Nichols

1. Ki=0, Kd=0 → Tăng Kp → Dao động đều

2. Kp = 0.6×Ku, Ki = 2Kp/Tu, Kd = Kp×Tu/8

3. Fine-tune qua Web Interface real-time

Encoder Odometry

Chu vi bánh = π × 65mm ≈ 204.2 mm
960 xung/vòng → Mỗi xung ≈ 0.213 mm
